# homebrew-tap

Homebrew tap for [Prosper](https://github.com/ventsislav-georgiev/prosper).

```sh
brew install --cask ventsislav-georgiev/tap/prosper
```

The cask is auto-published by Prosper's release workflow on every `v*` tag.
Prosper also updates itself in-app via Sparkle; `auto_updates true` keeps the
two from conflicting.
